1. 添加到本地索引
git add .
2. 添加到本地库
git commit -m "XXX"
3. 推送到远程库
git push
4. 取到本地库
git pull
5. 克隆本地库
git clone
6. 取回所有branch
git branch fetch
7. 查看全部branch
git branch -a
8. 查看本地branch
git branch -r
9. 切换branch XXX
git checkout -b XXX origin/XXX
10. 切换回branch master
git checkout master
